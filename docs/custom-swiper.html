Swiper

Using Swipper plugin, you can easily create carousels. It's being used in almost all the pages where we are having slider in hero element.

Simple Example

Just specify data attribute data-toggle="swiper" to any div element containing your items to enabled it. Optionally you can pass all options mentioned in the plugin's official documentation via data attribute swiper E.g. data-swiper="{}".
